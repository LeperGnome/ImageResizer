# ImageResizer
 With this web server you can:
 - upload images _(via url or file)_
 - view images using different parameters:
    - width 
    - height
    - size  _(max size in bytes)_

### Required software:
- Redis
- Python 3.8+

### To run:
 - Create virtual environment _venv_ : `$ virtualenv venv` in project folder
 - Run `$ ./launch.sh`:
